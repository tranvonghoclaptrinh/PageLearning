Web Design Learning Journey - PAGE

📌 Project Overview

This repository serves as a personal archive for my Web Design course projects. It documents my progress, from fundamental HTML/CSS concepts to advanced responsive layouts and interactive components. Each module represents a milestone in my journey to becoming a proficient web developer.

🚀 Learning Milestones

The project is organized into weekly modules, each focusing on specific skills:

Week 1: Fundamentals of web structure (HTML) and styling (CSS).

Week 2: Working with Forms, Input validation, and UI components.

Week 3: User Authentication interfaces and University Portal layouts.

Lunar New Year Project: A comprehensive Landing Page integrating all learned techniques including Responsive Design and Dark Mode.

🛠️ Tech Stack

HTML5: Semantic structure.

CSS3 (Tailwind CSS): Modern styling, utility-first approach, and custom hover animations.

JavaScript (Vanilla JS): Dark mode logic and DOM manipulation.

FontAwesome: Vector icons and social links.
